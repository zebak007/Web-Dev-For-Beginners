# 练习 HTML：建立博客雏形

## 简介

想象你正在设计，或重新设计你的个人网站。建立网页的图形设计雏形，然后写下用于构建网站的各种 HTML 标记。你可以写纸上再扫描，或者用软件，只要确保是手写的。
