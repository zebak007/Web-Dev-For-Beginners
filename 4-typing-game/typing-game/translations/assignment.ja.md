# 新しいキーボードゲームを作成する

## 説明書

キーボードのイベントを使ってタスクを行う小さなゲームを作りましょう。異なる種類のタイピングゲームや、キー入力で画面にピクセルをペイントするアートタイプのゲームにすることもできます。創造力を発揮してください。

## ルーブリック

| 基準 | 模範的な例                | 適切な                 | 改善が必要 |
| -------- | ------------------------ | ------------------------ | ----------------- |
|          | フルゲームが発表される。 | ゲームは非常にミニマム。 | ゲームにはバグがある。 |
