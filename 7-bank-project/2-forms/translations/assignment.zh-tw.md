# 造型化你的銀行程式

## 簡介

建立新的檔案 `styles.css`，匯入到你的 `index.html` 檔案中。藉由 CSS 檔，你能讓*登入*與*儀表板*頁面看起來更漂亮且整潔。試著為你的程式加入主題色彩，對應到你的品牌。

> 提示：你可以修改 HTML 檔，在必要時新增元素與 classes。

## 學習評量

| 作業內容 | 優良                                               | 普通                                 | 待改進                                   |
| -------- | -------------------------------------------------- | ------------------------------------ | ---------------------------------------- |
|          | 所有的頁面整潔且易讀：統一的主題色彩且排版顯示正常 | 頁面有調整過，但缺乏主題且排版有瑕疵 | 頁面缺乏造型，排版凌亂且頁面資訊難以理解 |
