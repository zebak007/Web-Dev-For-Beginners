# 関数で楽しむ

## 説明書

何かを返す関数と何も返さない関数の両方の異なる関数を作成します。

デフォルト値を持つパラメータとパラメータが混在した関数を作成できるかどうかを見てみましょう。

## ルーブリック

| 基準 | 模範的な例                                                                              | 適切な                                                         | 改善が必要 |
| -------- | -------------------------------------------------------------------------------------- | ---------------------------------------------------------------- | ----------------- |
|          | ソリューションは、多様なパラメータを持つ2つ以上の関数で提供されます。 | ワーキングソリューションは、1つの関数と少数のパラメータで提供されます。 | ソリューションにはバグがあります。 |
