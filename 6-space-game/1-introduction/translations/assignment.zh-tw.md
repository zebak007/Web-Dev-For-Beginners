# 建立遊戲雛形

## 簡介

使用本課程中的程式案例，編寫一款你喜歡的遊戲呈現方式。這是一款簡單小規模的遊戲，目的是要能以 class、組合模式與發布訂閱模式呈現遊戲的運作方式。發揮你的創意！

## 學習評量

| 作業內容 | 優良                       | 普通                       | 待改進                       |
| -------- | -------------------------- | -------------------------- | ---------------------------- |
|          | 畫面上有三個元素且能被控制 | 畫面上有兩個元素且能被控制 | 畫面上只有一個元素且能被控制 |
