# 為你的程式做註解

## 簡介

打開遊戲資料夾中目前的 /app.js 檔案，試著幫它做上註解並整理乾淨。程式碼很容易脫離掌控，現在是個好機會來確保你的程式是容易去閱讀的，在未來還可以被使用。

## 學習評量

| 作業內容 | 優良                          | 普通                    | 待改進                  |
| -------- | ----------------------------- | ----------------------- | ----------------------- |
|          | `app.js` 完整地註解且分塊整理 | `app.js` 有做充分的註解 | `app.js` 凌亂且缺乏註解 |
